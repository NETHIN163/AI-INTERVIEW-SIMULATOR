# ProPrep Professional Interview Simulator

ProPrep is a high-fidelity interview simulation platform designed for modern professionals to master technical and behavioral interviews. Built with a focus on performance, aesthetics, and realistic role-playing.

## Key Features
- **Company-Specific Challenges**: Dynamic question sets from top-tier companies including Google, Amazon, Meta, and more.
- **Role-Tailored Sessions**: 15-question dynamic assessments customized for Frontend, Backend, Data Science, PM, and other specialized roles.
- **Matte Design System**: A sophisticated, non-intrusive UI built for concentration and clarity.
- **Expert Analytics**: Comprehensive scoring and improvement roadmaps based on industry standards.

## Tech Stack
- **Framework**: Next.js 15+ (App Router)
- **Engine**: React 19
- **Database**: Repository-based dynamic JSON storage
- **Styling**: Vanilla CSS with Matte System tokens

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) to start your first session.
